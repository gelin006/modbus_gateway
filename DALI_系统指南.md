# DALI 智能照明系统 — 布线、安装与调试完整指南

---

## 第一章：DALI 系统概述

### 1.1 什么是 DALI？

DALI（Digital Addressable Lighting Interface，数字可寻址照明接口）是国际标准 **IEC 62386** 定义的专为照明设备设计的数字通信协议。与传统的 0-10V 模拟调光不同，DALI 采用**数字寻址**方式，每个灯具、驱动器、传感器都是独立的智能节点，通过一对双绞线组成总线网络。

### 1.2 DALI 系统核心特性

| 特性 | 说明 |
|------|------|
| **数字寻址** | 每个设备有唯一短地址（1-64），可单独控制 |
| **双向通信** | 不仅发送指令，还能读取灯具状态、故障反馈 |
| **分组控制** | 最多 16 个分组（Group），灵活组合 |
| **场景控制** | 最多 16 个场景（Scene），一键切换 |
| **总线供电** | DALI 总线自带 16V DC 电源，控制信号与供电共用线路 |
| **拓扑灵活** | 支持星型、树型、线型（手拉手），不限制极性 |

### 1.3 DALI 与常见协议对比

| 项目 | DALI | 0-10V 调光 | KNX | DMX512 |
|------|------|-----------|-----|--------|
| 通信方式 | 数字双向 | 模拟单向 | 数字总线 | 数字差分 |
| 单线设备数 | 64 个 | 无限制（同信号） | 无限制 | 32 单元 |
| 寻址能力 | ✅ 单灯/组/广播 | ❌ 无法单独 | ✅ 丰富 | ✅ 通道级 |
| 状态反馈 | ✅ 支持 | ❌ 不支持 | ✅ 支持 | ❌ 不支持 |
| 布线要求 | 普通 2 芯线 | 屏蔽线 | 专用总线 | 5 芯 XLR |
| 适用场景 | **商业照明** | 简单调光 | 楼宇控制 | 舞台灯光 |

## 第二章：DALI 系统硬件组成

### 2.1 系统架构图

```
                   ┌────────────────┐
                   │   DALI 控制器   │  ← 主控/网关，连接总线与上位机
                   │  (Master/Gateway)│
                   └───────┬────────┘
                           │ DALI 总线 (2芯, 16V DC)
          ┌────────────────┼────────────────────┐
          │                │                    │
    ┌─────┴─────┐   ┌─────┴─────┐       ┌─────┴─────┐
    │ DALI 驱动  │   │ DALI 驱动  │  ...  │ DALI 驱动  │  ← LED/荧光灯驱动
    │   (1-64)   │   │   (1-64)   │       │   (1-64)   │
    └─────┬─────┘   └─────┬─────┘       └─────┬─────┘
          │                │                    │
      ┌───┴───┐        ┌───┴───┐           ┌───┴───┐
      │ LED灯 │        │ LED灯 │   ...     │ LED灯 │
      └───────┘        └───────┘           └───────┘

     ┌──────────┐   ┌──────────┐      ┌──────────┐
     │ DALI面板 │   │  DALI传感器 │ ... │ DALI调光器│  ← 输入/控制设备
     └──────────┘   └──────────┘      └──────────┘
```

### 2.2 主要设备类型

#### 2.2.1 DALI 控制器（Master/Gateway）
- 系统核心，管理总线通信、地址分配、场景存储
- 常见形式：DALI USB 网关（连接 PC 调试软件）、IP 网关（接入网络）
- 每台控制器最多管理 **1 条总线 × 64 个地址**
- 部分控制器支持双总线：**2 × 64 = 128 个地址**

#### 2.2.2 DALI 驱动器（Driver/Ballast）
- 将 DALI 数字信号转换为 LED/荧光灯的调光/调色控制
- **DT6 类型**：单通道调光（仅亮度）
- **DT8 类型**：多通道调色（双色温 TC / RGBW / xy 色坐标）
- 支持设备类型：DT0（荧光灯）、DT6（LED 调光）、DT8（LED 调色调光）

#### 2.2.3 DALI 输入设备
- **控制面板**：场景切换面板、调光面板（通常为 4/6/8 键）
- **传感器**：移动传感器、照度传感器、组合传感器
- **调光器**：旋钮式或触摸式调光控制器

#### 2.2.4 DALI 总线电源（DALI PS）
- 为 DALI 总线提供 16V DC 电源
- 最大输出电流：250mA
- 通常集成在控制器内，也有独立模块

### 2.3 DT6 vs DT8 对比

| 对比项 | DT6 | DT8 |
|--------|-----|-----|
| **控制能力** | 仅亮度调光 | 亮度 + 色温/RGBW/xy |
| **占用地址** | **每路输出 = 1 个地址** | 1 个地址 = 多路输出 |
| **双色温方案** | 需 2 个地址（冷/暖各 1） | 1 个地址即可 |
| **RGBW 方案** | 需 4 个地址 | 1 个地址 |
| **成本** | 较低 | 略高 |
| **调试复杂度** | 简单 | 稍复杂 |
| **适用场景** | 单色温调光 | 氛围照明、可变色温 |

## 第三章：DALI 系统布线

### 3.1 线缆要求

DALI 对线缆要求**非常宽松**，这是其优势之一：

| 参数 | 要求 |
|------|------|
| **线缆类型** | 普通铜芯线即可，**无需屏蔽、无需双绞** |
| **推荐线径** | **1.5mm² ~ 2.5mm²**（建筑电气常用电源线） |
| **极性** | **不分极性**（DALI+ 和 DALI- 可任意连接） |
| **线缆材质** | 多股软线（穿管方便）或单股硬线（户外/工地） |

> ⚠️ **不要使用屏蔽双绞线！** 屏蔽层会增加分布电容，反而影响 DALI 总线信号质量。

### 3.2 最大传输距离

DALI 标准对总线压降有严格要求：**总线两端压降不得超过 2V**。

| 线径 | 推荐最大距离 | 备注 |
|------|------------|------|
| 1.0mm² | ~200m | 保守估算 |
| **1.5mm²** | **~300m** | 工程推荐标准 |
| 2.5mm² | ~400m+ | 长距离场景 |

> 若距离超过 300m，需加装 **DALI 中继器/放大器**。

### 3.3 总线容量限制

| 参数 | 上限 | 说明 |
|------|------|------|
| **最大设备数** | **64 个地址** | 每个逻辑设备占用 1 个短地址 |
| **总线最大电流** | **250mA** | DALI PS 输出能力 |
| **每条总线最大长度** | **300m** | 基于 1.5mm² 线缆 |

### 3.4 允许的拓扑结构

```
✅ 线型（手拉手）：      控制器 ── A ── B ── C ── D
✅ 星型：               控制器 ── A
                        ├── B
                        ├── C
                        └── D
✅ 树型：               控制器 ── A ── B
                        └── C ─── D
                                └── E
❌ 环形：               控制器 ── A ── B
                        └─────────────┘
```

> **严禁环形（Loop）接线！** 环形会导致信号反射和通信故障。

### 3.5 布线实践要点

#### ✅ 正确做法
- DALI 线与强电线（220V AC）**保持 ≥ 30cm 间距**
- 若必须交叉，**垂直交叉**而非平行走线
- 总线末端建议预留线路长度，方便后期增加设备
- 每个支路长度按"最远两端设备"之间的距离计算

#### ❌ 常见错误
- ❌ DALI 线与强电**平行敷设过长** → 电磁干扰导致丢包
- ❌ 使用**屏蔽双绞线** → 分布电容大，信号畸变
- ❌ **环形接线** → 信号反射使通信中断
- ❌ 线径**过细（<1.0mm²）** → 压降超标，设备掉线
- ❌ 总线**超过 64 个设备** → 地址不足，冲突

### 3.6 接线示意图

#### DALI 驱动接线（以 LED 驱动为例）

```
            ┌───────────────────┐
  220V AC ──┤ L   N  (电源输入) │─── LED + (正极)
            │                   │─── LED - (负极)
            │                   │
DALI+ ──────┤ DA+               │
DALI- ──────┤ DA- (DALI 输入)   │
            └───────────────────┘
```

#### 控制面板接线

```
            ┌───────────────────┐
DALI+ ──────┤ DA+               │
DALI- ──────┤ DA-               │
            │                   │  按钮 1 ── 场景 1
            │                   │  按钮 2 ── 场景 2
            │    面板控制器      │  按钮 3 ── 调光 +
            └───────────────────┘  按钮 4 ── 调光 -
```

## 第四章：DALI 系统安装

### 4.1 安装前准备

1. **图纸核对**：
   - 总线拓扑图（确定线型/星型/树型结构）
   - 每条总线设备数量（≤ 64 个）
   - 传感器与灯具位置匹配（覆盖范围合理）

2. **设备检查**：
   - 所有驱动器、面板、传感器是否为 DALI-2 认证
   - DT6/DT8 类型确认（影响地址分配策略）
   - 供电电压检查（220V AC ±10%）

3. **工具准备**：
   - DALI 调试软件 + USB 网关（如 Dalitools + VH-DLGW-U）
   - 万用表（测量总线电压 16V DC ±2V）
   - 网络寻线仪（用于线缆标识）

### 4.2 安装步骤

#### 第一步：DALI 驱动器安装
1. DALI 驱动器安装在**强电配电箱**内
2. 预留足够导轨空间
3. 所有控制回路负载接入 DALI 驱动器输出端
4. 驱动器供电：AC 220V ±10%

#### 第二步：DALI 总线布线
1. 按设计拓扑布线（避免环形）
2. 线径 ≥ 1.5mm²
3. 总线末端做好标记 DA+ / DA-
4. 总线与强电保持 ≥ 30cm 间距

#### 第三步：控制面板安装
1. 面板距地 **1.3 米高度**
2. 采用标准 **86 底盒**安装
3. 面板箭头方向向上
4. 面板后网口采用 **手拉手** 串联

#### 第四步：传感器安装
1. 移动传感器：避开门窗、空调出风口
2. 照度传感器：避免阳光直射
3. 传感器高度按产品手册要求（通常 2.5-4m）

#### 第五步：总线供电检查
1. 上电后测量 DALI 总线电压：**16V ±2V DC**
2. 检查总线极性（虽然不区分，但统一标记方便排查）
3. 确认总线电流 ≤ 250mA

### 4.3 安装检查清单

| 检查项 | 标准 | 确认 |
|--------|------|------|
| 总线设备数 | ≤ 64 个 | □ |
| 总线长度 | ≤ 300m | □ |
| 线径 | ≥ 1.5mm² | □ |
| 拓扑结构 | 非环形 | □ |
| 总线电压 | 16V ±2V | □ |
| 总线电流 | ≤ 250mA | □ |
| DALI 与强电间距 | ≥ 30cm | □ |
| 面板安装高度 | 1.3m | □ |
| 驱动器供电 | 220V AC | □ |

## 第五章：DALI 系统调试

### 5.1 调试流程概览

```
通电检查 → 物理连接确认 → 安装调试软件
    ↓
自动扫描 → 寻址与地址分配
    ↓
设备参数配置 → 分组 → 场景设置
    ↓
功能测试 → 单灯 → 分组 → 场景联动
    ↓
传感器校准 → 参数微调
    ↓
交付验收
```

### 5.2 调试软件与环境

#### 常用调试软件
- **Dalitools**：元皓科技 DALI 调试软件，广泛使用
- **DALI Cockpit**：DiiA 官方调试工具
- 各厂商专用调试软件（如欧司朗、飞利浦、锐高等）

#### 硬件要求
| 设备 | 说明 |
|------|------|
| 电脑 | Windows XP/Win7/Win10/Win11 |
| DALI USB 网关 | 如 VH-DLGW-U、CTT-DLM-1 |
| USB 转 RS485 转换器 | 推荐 PL2303 芯片 |
| DALI 总线电源 | 16V DC，≥250mA |

#### 连接方式
```
电脑 (USB) → USB转RS485 → DALI USB网关 → DALI 总线
```

### 5.3 自动扫描与寻址

#### 5.3.1 自动扫描
1. 打开调试软件，选择对应控制器
2. 点击 **"自动扫描 / Auto Scan"**
3. 软件将识别总线上所有 DALI 设备

#### 5.3.2 地址分配
- 扫描到的设备默认显示为 **"未编址"**（短地址 255）
- 执行**自动分配地址（Auto Address）**：
  - 系统按顺序分配短地址 1~64
  - 地址以物理随机方式分配（非顺序，避免布线位置猜忌）

- 手动分配：支持自定义短地址（地址 0 ~ 63）

#### 5.3.3 识别设备
- 可通过 **"识别 / Identify"** 功能让指定灯具闪烁，确认物理位置
- 建议**实际标记**每个地址对应的灯具位置（贴标签或记录表格）

#### 地址分配记录表示例

| 短地址 | DT 类型 | 物理位置 | 分组 | 备注 |
|--------|---------|---------|------|------|
| 1 | DT6 | 会议室-主灯 | 1 | 调光 |
| 2 | DT6 | 会议室-筒灯1 | 1 | 调光 |
| 3 | DT6 | 会议室-筒灯2 | 1 | 调光 |
| 4 | DT8 | 会议室-氛围灯带 | 2 | 双色温 |
| ... | ... | ... | ... | ... |

### 5.4 DT6 与 DT8 调试差异

#### DT6 调试
- 扫描后显示为 **"DT6 LED 灯具 / Led lamp"**
- 仅支持亮度调节（0~100%）
- 每个输出 = 1 个短地址
- 需分别设置：**功率限制、调光曲线、调光下限、淡入淡出时间**

#### DT8 调试
- 扫描后显示为 **"多类型 / MultiType"**
- 1 个地址支持多通道控制（亮度 + 色温/RGBW/xy）
- **TC（Tunable White）**：通过色温窗口控制
- **RGBW**：通过颜色选择窗口控制
- **xy 色坐标**：通过色度图控制

### 5.5 分组配置

DALI 支持 **16 个分组**（Group 0~15），可按区域或功能灵活组合：

| 分组 | 包含设备 | 用途 |
|------|---------|------|
| Group 0 | 广播（所有设备） | 全开/全关 |
| Group 1 | 1~3 | 会议室主照明 |
| Group 2 | 4 | 会议室氛围灯带 |
| Group 3 | 5~8 | 走廊灯 |
| Group 4 | 9~12 | 工位区域 |
| ... | ... | ... |

> 一个设备**可以同时属于多个分组**。

### 5.6 场景设置

DALI 支持 **16 个场景**（Scene 0~15），每个场景可存储所有设备的亮度/色温值：

| 场景 | 名称 | 效果 |
|------|------|------|
| Scene 1 | 会议模式 | 主灯 100%，氛围灯 50%，筒灯 80% |
| Scene 2 | 投影模式 | 主灯 20%，氛围灯 30%，筒灯 10% |
| Scene 3 | 休息模式 | 主灯 50%，氛围灯 80%（暖光），筒灯 30% |
| Scene 4 | 清洁模式 | 所有灯 100% |

**场景配置流程：**
1. 逐灯调节到目标亮度/色温
2. 选择 **"存储为场景 / Store Scene"**
3. 指定场景编号（0~15）
4. 分配到控制面板对应按键

### 5.7 功能测试清单

| 测试项目 | 方法 | 正常结果 |
|---------|------|---------|
| **单灯控制** | 选中单个地址，调节亮度 | 亮度 0~100% 平滑变化 |
| **组控制** | 选中分组，发送指令 | 组内所有灯具同步响应 |
| **广播控制** | 发送广播指令 | 总线所有设备响应 |

### 5.8 常见调试问题及排查

| 问题 | 可能原因 | 解决方案 |
|------|---------|---------|
| **扫描不到设备** | 总线无供电 | 测量总线电压（需 16V） |
| | 连接线错误 | 检查 USB 网关到 DA+ DA- 接线 |
| | 设备地址冲突 | 重置所有设备（发送清除地址指令） |
| **部分灯具无响应** | 地址冲突 | 重新分配地址 |
| | 总线压降过大 | 测量末端电压，<14V 则加中继器 |
| | 线路过长/线径细 | 换粗线或加中继器 |
| **通信不稳定** | 强电干扰 | 检查 DALI 线与强电间距 |
| | 总线环形 | 检查拓扑，切断环形 |
| **调光闪烁** | 调光下限过低 | 提高最小调光值（建议 1~3%） |
| | 驱动与灯具不匹配 | 核对驱动额定功率与灯具 |
| | 电网谐波 | 加滤波电容/换抗干扰驱动 |
| **DT8 颜色异常** | DT6/DT8 混淆 | 确认驱动类型，DT8 需专用设置 |
| | TCP 参数错误 | 检查色温校正偏移值 |
| **面板按键无效** | 面板未编址 | 面板也需要 DALI 短地址 |
| | 场景未存储 | 检查面板按键对应的场景号 |

### 5.9 高级调试参数

#### 调光参数（可通过调试软件配置）

| 参数 | 范围 | 默认值 | 说明 |
|------|------|--------|------|
| **Min Level** | 0.1%~100% | 3% | 最低亮度值 |
| **Max Level** | 0.1%~100% | 100% | 最高亮度值 |
| **Fade Time** | 0~15 | 1 | 淡入淡出时间（0=立即，1≈0.7s...15≈90min） |
| **Fade Rate** | 1~15 | 3 | 调光速率 |
| **Power On Level** | 0~254/255 | 254 | 上电后亮度（254=记忆上次） |
| **System Failure Level** | 0~254/255 | 254 | 总线故障时亮度 |

#### 传感器参数

| 参数 | 说明 |
|------|------|
| **检测延时** | 检测到移动后多久响应（0~5 分钟） |
| **关闭延时** | 无移动后多久关闭（30 秒~30 分钟） |
| **照度阈值** | 当环境照度低于此值时触发 |
| **保持时间** | 保持当前状态的时间 |
| **灵敏度** | 0~100%，调整探测器响应 |

## 第六章：系统交付与维护

### 6.1 验收清单

| 验收项 | 标准 |
|--------|------|
| 设备清单 | 所有设备地址记录完整，与实际物理位置对应 |
| 功能验证 | 单灯、分组、场景、定时功能全部通过 |
| 通信稳定 | 连续运行 24 小时无通信故障 |
| 应急功能 | 断电恢复后正确回到预设状态 |
| 文档交付 | 提供设备布局图、地址表、场景配置表 |

### 6.2 日常维护

- 定期**备份** DALI 控制器中的配置（地址、分组、场景）
- 更换灯具时，确保新驱动使用**相同 DALI 短地址**
- 发现通信异常时优先检查**总线电压**（16V ±2V）
- 定期**清除总线设备列表**并重新扫描（如有设备更换）

### 6.3 扩展注意事项

- 总线设备数不足 64 时可随时添加新设备
- 修改分组/场景**无需重新布线**，仅通过软件即可
- 超过 64 个设备时，需增加**第二条 DALI 总线**或增加控制器
